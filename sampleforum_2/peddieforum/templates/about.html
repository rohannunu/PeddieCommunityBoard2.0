About the Website

Description

Using this website, Peddie students can learn about the different clubs at Peddie, as well as their meeting times. Not only does this website reduce all school emails, but having a place where students can learn more about the clubs at Peddie provides an immersive experience to all that Peddie has to offer in the extracurriculars.

Goals

This website is to help Peddie students, whether remote or in-person, connect with each other and learn more about Peddie's culture.
